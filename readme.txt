Displays author specific archives widget on author's single post page.

== Description ==
Simple plugin helps to personalize authors or users page with sidebar widgets specific to authors or users page. It will display author specific archives.
Features:
1. Custom widget title, option to add author name.
2. Counts number of posts by author.
3. Easy to add.



== Installation ==

1. Upload `Author Archives` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The configuring options are in the usual widget admin interface.

== Frequently Asked Questions ==

= How to configure Author Archives widget? =
Follow below steps:
1. Enable the plugin from plugin area.
2. Go to Appearance-> Widget; select the "Author Archives" widget and drag in the sidebar.
3. Configure the title text. 

= How to configure author's name in widget title? =
In the title text add [authorname] shortcode.

= Where does this Widget appear? =
This widget appears on single post page.
